# Session Notes

Development session logs for footnotes.at. Each session documents completed work, technical decisions, and context for future reference.

## Format

Each session file follows this structure:

```markdown
# Session Notes - YYYY-MM-DD

## Session Summary
Brief overview of what was accomplished.

## Completed Tasks
Detailed list of completed work with file references.

## Technical Notes
Implementation details, decisions, gotchas.

## Pending/Discussed
Items mentioned but not completed.

## Git Commits
List of commits from the session.
```
